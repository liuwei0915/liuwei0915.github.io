For the server Jump proxy:

Step 1: Create public and private keys

The config file is stored in the path: /Users/weiliu/.ssh

    <ssh-key-gen on local-host such as ~/.ssh>

Step 2: Copy the public key to remote-host using ssh-copy-id

	<ssh-copy-id -i ~/.ssh/id_rsa.pub remote-host>

for example:

    <ssh-copy-id -i ~/.ssh/id_ed25519.pub wl654632@ssh.csun.edu>

remote host could be like user_ID@ssh.sandbox.csun.edu

Step 3: Login to remote-host without entering the password

ssh remote-host

command using in bash

  1. command > file: Send the stdout of command to file.
  2. command 1> file: Send the stdout of command to file. Since stdout is FD 1, that’s the number we put in front of the redirection operator. This is identical to the previous example, because FD 1 is the default for the > operator.
  3. command < file: Use the contents of file when command reads from stdin.
  4. command 0< file: Use the contents of file when command reads from stdin, exactly as in the previous example, since FD 0 (stdin) is the default for the < operator.
