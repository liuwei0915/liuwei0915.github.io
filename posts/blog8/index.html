variable

  • let is the preferred way to declare a variable when it can be reassigned
    const is the preferred way to declare a variable with a constant value.
    • variable could be used directly in the string:
    let name =  wei  (or const name = wei )  
console.log(name) // return wei

    const variables cannot be changed after they’re created.

    • String Interpolation:
    let name = wei;
console.log(`what is your ${name}`) // output what is your wei. in the string of log(), you have to use ${}to output the variable name. and use backtick "`" to quote the string.

    let age = 7;
// String concatenation
'Tommy is ' + age + ' years old.';

// String interpolation
`Tommy is ${age} years old.`;
    • typeof operator:
    const unknown1 = 'foo';
console.log(typeof unknown1); // Output: string

Condition

  • If…Else statement:
let sale = true;
sale = false;
if(sale) {
  console.log('Time to buy!');
} else {
  console.log('Time to wait for a sale.')
}
  • comparison operator:
let hungerLevel = 7;
// for comparison numbers >, <, <=,>=,==
// for string equal or not 
// String is equal to: ===
// string is not equal to: !==
if (hungerLevel >= 7) {
  console.log( 'Time to eat!');
} else {
  console.log('We can eat later!');
}
  • true or false assignment Assign to writingUtensil the value of tool and if tool is falsy, assign a default value of ‘pen’.
let tool = '';
let writingUtensil = tool || 'pen'
  • Ternary Operator / another way of if/else statement
isNightTime ? console.log('Turn on the lights!') : console.log('Turn off the lights!');

or

favoritePhrase === 'Love That!' ? console.log('I love that!'): console.log("I don't love that!");
