const squareNum = (num) => {

};

const squareNum = num => num * num;

Arrays

  • Arrays in JavaScript are zero-indexed, meaning the positions start counting from 0 rather than 1. Therefore, the first item in an array will be at position 0. Arrays are created with brackets [].
const hobbies = ['10', '2','3' ]
  • Update Elements
hobbies[1] = 'avocado' //it will change the second element value to avocado

const variable array can be mutate, but not reassign an new value. let could be

const numberArray = [0, 1, 2, 3];
numberArray[1] = 'ss'
console.log(numberArray);
  • .push() method
const itemTracker = ['item 0', 'item 1', 'item 2'];  
itemTracker.push('item 3', 'item 4');  
console.log(itemTracker);   
// Output: ['item 0', 'item 1', 'item 2', 'item 3', 'item 4'];
  • .pop() method
const newItemTracker = ['item 0', 'item 1', 'item 2'];  
const removed = newItemTracker.pop();  
//or newItemTracker.pop()
console.log(newItemTracker);   
// Output: [ 'item 0', 'item 1' ]
console.log(removed);  
// Output: item 2
  • more array method You can read about all of the array methods that exist on the [ Mozilla Developer Network (MDN) array documentation.] (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array) .shift() method to remove the first item of array .unshift method to add the item to the beginning of array
  • nested array
const nestedArr = [[1], [2, 3]];
 
console.log(nestedArr[1]); // Output: [2, 3]
console.log(nestedArr[1][0]); // Output: 2
